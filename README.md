# TicTacToe
A classic Tic-Tac-Toe game for two players. Players take turns on the same computer, using the mouse to mark the squares.

## Features
- Two-player mode on the same device
- Simple and intuitive interface
- Mouse click control

## Technologies Used
- C# / .NET Framework
- NAudio Library
